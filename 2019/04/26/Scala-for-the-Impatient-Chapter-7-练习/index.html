第4题

在你看来，Scala 的设计者为什么要提供package object语法而不是简单地让你将函数和变量添加到包中呢？

从封装性的角度把工具函数或常量添加到包而不是某一个对象更加合理，但是非常可惜 JVM 不支持，于是 Scala 的设计者提供了 package object 这样的语法来解决这个问题的局限性。
