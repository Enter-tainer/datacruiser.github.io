  • 链表中所有节点的值都是唯一的。

解题思路

这道题与常规的链表节点删除问题有些不同, 常规的操作都是给出一个头指针和待删除节点的值，然后通过遍历链表找到待删除节点，将待删除的节点的前一个节点的next指向待删除节点的next，然后将节点删除。

本题仅仅给出一个节点，需要一些非常规的方法。具体思路是这样的，将该节点的后一个节点的值赋值给该节点，然后将待删除节点的后一个节点删除，这样“曲线救国”实现删除当前节点的目的。实际上只是修改了当前节点的值，然后将与当前节点值相同的后一节点删除。具体代码如下：

代码

/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     struct ListNode *next;
 * };
 */
void deleteNode(struct ListNode* node) {
    node->val = node->next->val;
    node->next = node->next->next;
}
